Objetivo general: Mostrar la programación y los elementos necesarios para realizar un sitio web dinámico con lenguaje de programación y bases de datos.
Objetivo específico 1: Implementar una correcta semántica en lenguaje de marcado HTML e incorporar lenguaje de programación PHP para dinamizar el contenido.
Objetivo específico 2: Realizar consultas SELECT, INSERT, UPDATE y DELETE en lenguaje SQL con el fin de incorporar información en el sitio web.
Objetivo específico 3: Realizar consultas mediante formularios web con lenguaje de programación PHP y lenguaje de consulta MySQL.


SECCION A (1): En necesario realizar una contextualización del tema a manera global, explicar usos y aplicaciones del lenguaje de programación PHP así mismo su compatibilidad con sistemas web y utilidades dinámicos en conjunto con lenguaje de consulta MySQL.
SECCION B(2): Procesos y procedimientos para implementar formularios web interrelacionados con lenguajes de programación y de consulta, de igual manera con un sistema de marcado HTML. Personalización de un sitio web con ayuda de lenguajes de programación e información de una base de datos.
SECCION C(3): Contextualizar al usuario con el sistema de consultas a bases de datos mediantes una correcta sintaxis en lenguaje de consulta MySQL, y sistema de programación PHP. con MySQL Selecciona, inserta, actualiza o elimina información de una base de datos, con ayuda de una correcta sintaxis con el motor más utilizado a nivel mundial en sitios web.
